# TechStore Design System

## Typography Scale
```
H1: text-3xl (30px) - Product titles, main headings
H2: text-2xl (24px) - Section headings, feature cards
H3: text-lg (18px) - Accordion titles, subheadings
Body: text-base (16px) - Main content
Small: text-sm (14px) - Secondary info, table data
Tiny: text-xs (12px) - Breadcrumbs, meta info
```

## Spacing Scale
```
Section Vertical: py-16, py-32
Section Horizontal: px-8
Container Width: w-[75%] mx-auto
Gaps: gap-2 (8px), gap-4 (16px), gap-8 (32px), gap-12 (48px)
```

## Color Palette
```
Primary Text: text-gray-900 (#111827)
Secondary Text: text-gray-700 (#374151)
Tertiary Text: text-gray-600, text-gray-500
Borders: border-gray-200 (#E5E7EB)
Backgrounds: bg-white, bg-gray-50, bg-gray-100
Accents: Purple/Blue hover states
```

## Interactions
```
Transitions: duration-300, duration-500, ease-in-out
Hover Scale: hover:scale-110
Focus Rings: ring-1, ring-2, ring-offset-2
```

## Component Patterns
```
Cards: border border-gray-200 rounded-lg hover:shadow-lg
Buttons: border border-gray-200 hover:bg-gray-50 transition-colors
Inputs: border-gray-300 focus:ring-2 focus:ring-blue-500
Accordions: border-b border-gray-200 py-5
```
